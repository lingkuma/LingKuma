## 🌟 核心功能

### 📚 智能高亮與翻譯

| 功能 | 說明 |
| :--- | :--- |
| **多單字高亮** | 陌生單字藍色高亮,熟悉單字黃色高亮,生詞熟詞一目了然 |
| **句子爆炸** | 多個生詞同時查詢,附整句 AI 翻譯,快速理解複雜句子 |
| **單字筆記** | 記錄自己的理解和記憶,個性化學習筆記 |
| **AI 推薦筆記** | 句中解析,精確易懂,提供智能學習建議 |
| **AI 例句解析** | 可持續提問,深入理解單字用法和語法結構 |
| **例句記錄** | 重要句子不再忘記,隨時複習回顧 |
| **自訂外部按鈕** | 詞典百科一鍵搜尋,快速擴展學習資源 |

---

### 🌍 全平台支援

::: tip 跨平台無縫體驗
- **瀏覽器擴充功能**: Chrome / Edge / Firefox
- **行動端**: iOS / Android 全覆蓋
- **同步方式**: WebDAV / Docker 免費同步
- **閱讀器支援**: Readest 等 EPUB 網頁閱讀器
- **PDF 支援**: Mozilla PDF 閱讀器完美支援
:::

---

### 🎯 更多實用功能

#### YouTube 字幕增強
- **字幕控制**: 靈活控制字幕顯示和隱藏
- **AI 修復**: 無標點符號英文字幕智能修復
- **高亮支援**: Trancy 等外掛高亮相容

#### 閱讀輔助
- **Bionic 閱讀**: 提升閱讀速度和理解力
- **簡悅閱讀模式**: 高亮支援,專注閱讀體驗
- **液態玻璃彈窗**: 優雅的介面互動

#### 語音合成
- **自訂 URL 入口**: 靈活配置 TTS 服務
- **Edge TTS**: 高品質語音合成
- **Minimax TTS**: 多樣化語音選擇
