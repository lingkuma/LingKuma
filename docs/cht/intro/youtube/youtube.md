# YouTube 字幕使用教學

## 沉浸式字幕（電腦/手機支援）

1. 手動選擇目標語言的字幕，並啟動預設字幕。
    - 更換語言、翻譯等操作請在預設的字幕設定內更改，然後啟動即可。
2. 點擊右側快捷開關後即可啟動
