# YouTube 字幕使用教學

>[!tip]
電腦端和安卓端 lingkuma 已經兼容原版 trancy。手機端必須使用內置的 lingkuma trancy，因為 safari 似乎有些問題，導致無法在 trancy 的元素上做高亮等。如果你知道如何修改請在 issues 中分享，萬分感謝。

## 沉浸式字幕（電腦/手機支援）

1. 手動選擇目標語言的字幕，並啟動預設字幕。
    - 更換語言、翻譯等操作請在預設的字幕設定內更改，然後啟動即可。
2. 點擊右側快捷開關後即可啟動
