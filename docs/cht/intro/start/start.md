## 如何縮放視窗？

縮放算法是這裡的值 `1 / (頁面DPR/Custom DPR)` 2K 屏幕的頁面 DPR 是 2 ； Iphone 的 DPR 是 3；某些手機是 2 或者 1.8；所以可以用來調節視窗異常，以及可以用來手動放大縮小。



![](<./assets/1758997347657.png>)



## 如何更換彈窗背景？

若自訂背景，請根據圖片顏色固定為黑主題或者白主題，否則對比度不夠，影響閱讀。
支援svg,png,gif,mp4等
