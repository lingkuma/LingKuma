## 🌟 核心功能

### 📚 智能高亮与翻译

| 功能 | 说明 |
| :--- | :--- |
| **多单词高亮** | 陌生单词蓝色高亮,熟悉单词黄色高亮,生词熟词一目了然 |
| **句子爆炸** | 多个生词同时查询,附整句AI翻译,快速理解复杂句子 |
| **单词笔记** | 记录自己的理解和记忆,个性化学习笔记 |
| **AI推荐笔记** | 句中解析,精确易懂,提供智能学习建议 |
| **AI例句解析** | 可持续提问,深入理解单词用法和语法结构 |
| **例句记录** | 重要句子不再忘记,随时复习回顾 |
| **自定义外部按钮** | 词典百科一键搜索,快速扩展学习资源 |

### 🌍 全平台支持

::: tip 跨平台无缝体验
- **浏览器扩展**: Chrome / Edge / Firefox
- **移动端**: iOS / Android 全覆盖
- **同步方式**: WebDAV / Docker 免费同步
- **阅读器支持**: Readest 等 EPUB 网页阅读器
- **PDF支持**: Mozilla PDF 阅读器完美支持

### 🎯 更多实用功能

#### YouTube 字幕增强
- **字幕控制**: 灵活控制字幕显示和隐藏
- **AI修复**: 无标点符号英文字幕智能修复
- **高亮支持**: Trancy 等插件高亮兼容

#### 阅读辅助
- **Bionic 阅读**: 提升阅读速度和理解力
- **简悦阅读模式**: 高亮支持,专注阅读体验
- **液态玻璃弹窗**: 优雅的界面交互

#### 语音合成
- **自定义URL入口**: 灵活配置TTS服务
- **Edge TTS**: 高质量语音合成
- **Minimax TTS**: 多样化语音选择
