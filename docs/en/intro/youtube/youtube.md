# YouTube Subtitles Tutorial

## Immersive Subtitles (Desktop/Mobile Support)

1. Manually select the target language subtitles and enable the default subtitles.
    - For changing language, translation, and other operations, please change them in the default subtitle settings, then enable them.
2. Click the quick toggle switch on the right to enable

    - ![](https://res.cloudinary.com/dux7dnl5b/image/upload/v1768465649/boehrg0hdvmus0lwefhf.png)
    - ![](https://res.cloudinary.com/dux7dnl5b/image/upload/v1768464607/c5ksuumy1jipwg2chaxc.png)

## Embedded Subtitles
1. Enable the `Embedded Subtitles` option in the popup and refresh the page.
