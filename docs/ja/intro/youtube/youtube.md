# YouTube 字幕使用チュートリアル

>[!tip]
PC版とAndroid版では、LingKumaは元のTrancyと互換性があります。モバイル版では、Safariに問題があり、Trancy要素でのハイライトなどの機能が動作しないため、内蔵のLingKuma Trancyを使用する必要があります。修正方法をご存知の場合は、issuesで共有してください。よろしくお願いします。

## イマーシブ字幕（PC/スマホ対応）

1. 目標言語の字幕を手動で選択し、デフォルトの字幕を有効にします。
    - 言語の変更、翻訳などの操作は、デフォルトの字幕設定内で変更してから有効にしてください。
2. 右側のクイックトグルスイッチをクリックして有効にします

    - ![](https://res.cloudinary.com/dux7dnl5b/image/upload/v1768465649/boehrg0hdvmus0lwefhf.png)
    - ![](https://res.cloudinary.com/dux7dnl5b/image/upload/v1768464607/c5ksuumy1jipwg2chaxc.png)

## 組み込み字幕
1. ポップアップで`組み込み字幕`オプションを有効にし、ページをリフレッシュします。
    - ![](https://res.cloudinary.com/dux7dnl5b/image/upload/v1768465893/bkwhzeni1kcnjbte9yn9.png)
