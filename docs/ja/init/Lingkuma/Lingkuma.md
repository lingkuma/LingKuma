## 🌟 コア機能

### 📚 スマートハイライトと翻訳

| 機能 | 説明 |
| :--- | :--- |
| **複数単語ハイライト** | 未知の単語は青、馴染みのある単語は黄色、一目瞭然 |
| **文章爆発** | 複数の未知語を同時に検索、全文AI翻訳付き |
| **単語ノート** | 自分の理解を記録、パーソナライズされた学習ノート |
| **AI推奨ノート** | 文章解析、正確で分かりやすい |
| **AI例文解析** | 継続的な質問で単語の使い方と文法を深く理解 |
| **例文記録** | 重要な文章を忘れず、いつでも復習可能 |
| **カスタム外部ボタン** | 辞書・百科事典をワンクリックで検索 |

---

### 🌍 マルチプラットフォーム対応

::: tip シームレスなクロスプラットフォーム体験
- **ブラウザ拡張**: Chrome / Edge / Firefox
- **モバイル**: iOS / Android 完全対応
- **同期方法**: WebDAV / Docker 無料同期
- **リーダー対応**: Readest などの EPUB ウェブリーダー
- **PDF対応**: Mozilla PDF リーダー完全対応
:::

---

### 🎯 その他の実用的な機能

#### YouTube 字幕強化
- **字幕制御**: 字幕の表示・非表示を柔軟に制御
- **AI修復**: 句読点のない英語字幕をスマート修復
- **ハイライト対応**: Trancy などのプラグインと互換

#### 読書支援
- **Bionic 読書**: 読書速度と理解力を向上
- **簡悅読書モード**: ハイライト対応、集中した読書体験
- **液体ガラスポップアップ**: エレガントなインターフェース

#### 音声合成
- **カスタム URL エントリー**: 柔軟な TTS サービス設定
- **Edge TTS**: 高品質な音声合成
- **Minimax TTS**: 多様な音声オプション
